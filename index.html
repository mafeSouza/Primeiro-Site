O que é HTML

HTML, que significa Hypertext Markup Language, é a linguagem padrão para criação de páginas da web. Ele define a estrutura básica de uma página da web e usa uma série de elementos, tags e atributos para formatar o conteúdo.

Estrutura Básica

Uma estrutura HTML básica inclui as seguintes partes:

  1. html: Elemento raiz que engloba todo o conteúdo da página.
  2. head: Contém metadados e informações sobre a página.
  3. body: Contém o conteúdo principal da página, visível aos usuários.

Listas

HTML suporta dois tipos principais de listas: listas ordenadas e listas não ordenadas.

Listas Ordenadas

  1. Item 1
  2. Item 2
  3. Item 3

Listas Não Ordenadas

  • Item A
  • Item B
  • Item C
